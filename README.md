# Telegram-бот для маникюрного салона

## Описание бота

Профессиональный Telegram-бот для автоматизации записи клиентов в маникюрный салон. Бот позволяет клиентам самостоятельно выбирать удобное время, просматривать портфолио и прайс-лист, а мастеру — эффективно управлять расписанием через удобную админ-панель.

## Ключевые возможности

### Для клиентов:
- Календарь с выбором даты на месяц вперед с отображением только свободных дней
- Выбор доступного времени из списка свободных слотов
- Простая форма записи (имя и номер телефона)
- Просмотр портфолио с переходом на Pinterest-галерею
- Актуальный прайс-лист с ценами на услуги
- Возможность отмены своей записи
- Автоматическое напоминание за 24 часа до визита

### Для администратора:
- Защищенная админ-панель с доступом только по ID
- Управление расписанием (добавление рабочих дней и временных слотов)
- Возможность полностью закрыть определенную дату
- Просмотр всех записей на выбранную дату
- Отмена записей клиентов

## Система безопасности

- Обязательная проверка подписки на Telegram-канал перед записью
- Защита от множественной записи (один клиент не может записаться на несколько слотов)
- Автоматическое освобождение слота при отмене записи

## Уведомления

- Мгновенное оповещение мастера о новой записи
- Публикация информации о записи в отдельный Telegram-канал с расписанием
- Напоминание клиенту о предстоящем визите за 24 часа

## Технические особенности

- Python + aiogram 3.x (современная и надежная архитектура)
- SQLite (легкая база данных, не требует отдельного сервера)
- FSM (правильная обработка состояний пользователя)
- Inline-календарь (интуитивно понятный интерфейс)
- APScheduler (надежный планировщик напоминаний)

## Преимущества для бизнеса

- Экономия времени на записи клиентов
- Снижение количества неявок благодаря автоматическим напоминаниям
- Возможность записи 24/7 без участия администратора
- Современный сервис повышает лояльность клиентов
- Полный контроль над расписанием в одном месте

## Интерфейс

Все взаимодействие построено на inline-кнопках:
- Главное меню с основными разделами
- Календарь для выбора даты
- Сетка доступного времени
- Подтверждение записи
- Админ-панель с инструментами управления

## Для кого подойдет

- Частные мастера маникюра
- Маникюрные салоны
- Студии красоты
- Brow-бары
- Любые бьюти-услуги по записи

---

Бот полностью готов к использованию. Для запуска достаточно добавить токен бота, ID администратора и ID канала в файл конфигурации.
